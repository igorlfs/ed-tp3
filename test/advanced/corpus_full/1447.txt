nec spectraview ii color calibration kit  svii pro kit  designed for professionals with color critical applications  spectraviewii combines award winning nec display technology with a color measurement sensor and sophisticated calibration software  the result is highly accurate  reliable  repeatable  feature rich solution for display calibration and profiling  the spectraviewii system uses a color sensor to take color measurements of the display screen during calibration  the software analyzes these measurements and sends color adjustment commands directly to the display monitor  this means that color adjustments are made in the monitor rather than in the video graphics adapter  with spectraviewii  the video graphics adapter is not used at all to make any gamma or tone response curve corrections to the display  so the fidelity of the system is maintained
